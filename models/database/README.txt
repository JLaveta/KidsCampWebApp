-------------------------------------------------------------
NOTES ON USAGE

Step 1: Run the create_db.sql script on the server.
Step 2: Run the populate_db.sql to populate with dummy data 
		(optional).
Step 3: Run each of the .sql files in the stored_procs folder 
		to populate stored procedures.
-------------------------------------------------------------
NOTES ON BACKUP/RECOVERY SCRIPTS

Each script needs to be edited to include your database/serve 
information, including:

1. User
2. Password
3. Host
4. Database name
5. File path to store backups
